## 宏观经济

经济分三类，一类是微观经济，是搞管理研究的，叫管理学家；第二个是中观经济，就是搞企业研究、搞战略研究的；还有一类是宏观经济，是讲国家宏观政策、世界贸易、世界货币怎么样。

经济学的基本原理，就是专业化分工带来社会价值。
